# mypt

## Get started

- Install [nvm](https://github.com/nvm-sh/nvm) or [nodenv](https://github.com/nodenv/nodenv)
- `nvm use` or `nodenv install`
- `npm install`

## Develop

`npm run dev` starts a web server in watch mode on [http://localhost:8080](http://localhost:8080).

## Build

`npm run build`

## Deploy

- `firebase login`
- `npm run deploy`
